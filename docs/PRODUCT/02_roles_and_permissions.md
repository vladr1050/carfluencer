# Roles and Permissions

## Admin

Platform operator.

Capabilities:

- manage media owners
- manage advertisers
- manage vehicles
- manage campaigns
- manage pricing
- manage discounts
- manage telemetry devices
- configure data collection
- view analytics

---

## Media Owner

Owner of vehicles used for advertising.

Capabilities:

- register account
- add vehicles
- upload vehicle photos
- see earnings per vehicle
- see campaigns where vehicles participate
- upload campaign proof photos

---

## Advertiser

Company buying advertising campaigns.

Capabilities:

- register account
- view campaigns
- view vehicles
- see pricing
- see individual discounts
- view heatmaps
