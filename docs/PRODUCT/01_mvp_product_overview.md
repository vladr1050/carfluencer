# MVP Product Overview

## Product

Carfluencer x Evo.ad

A platform for selling advertising space on vehicles and visualizing exposure data.

Initially integrated with:

Carguru car sharing fleet.

---

## Key Concept

Advertisers buy advertising placements on cars.

Vehicles generate telemetry data.

This data is used to visualize:

- movement
- parking
- exposure zones

using maps and heatmaps.

---

## MVP Core Value

Provide **transparency of advertising exposure**.

Advertisers should see:

- where vehicles drove
- where they parked
- which areas had the highest visibility

---

## Core Features

### Inventory

Vehicles available for advertising.

---

### Campaigns

Advertisers run campaigns using selected vehicles.

---

### Telemetry

Vehicle GPS data imported from ClickHouse.

---

### Heatmaps

Visualization of:

Driving activity

Parking exposure

---

### Revenue tracking

Media owners see income generated by their vehicles.

---

## Users

Three main roles:

Admin  
Media Owner  
Advertiser

---

## Architecture

Admin panel → Laravel + Filament

User portals → React

Database → PostgreSQL

Telemetry source → ClickHouse
